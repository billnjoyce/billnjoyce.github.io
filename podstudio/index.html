HOW IT WORKS

정보만 넣으면, 나머지는 자동입니다

복잡한 인쇄 규격도, 디자인 툴도 필요 없어요. 4단계면 교보 POD 등록용 파일이 완성됩니다.

1

도서 정보 입력

한글·워드에서 “PDF로 저장” 후 그 파일을 올리세요.

2

원하는 판형 선택

신국판·A5·크라운판… 원하는 책 크기를 고르세요.

3

규격 자동 변환

어떤 크기 원고든 고른 판형에 딱 맞게 재구성.

4

바로 업로드

교보에 그대로 올리면 되는 내지 PDF 완성.

FEATURES

출판 초보도, 전문가처럼

교보문고 바로출판 POD의 까다로운 제작 가이드를 그대로 엔진에 담았습니다.

어떤 PDF든 · 크기 무관

A4로 썼든 레터 크기든, 한글·워드·구글독스 무엇으로 만든 PDF든 상관없어요. 어떤 크기의 PDF를 넣어도 고른 판형의 교보 공식 규격으로 모든 페이지를 통일합니다.

내용은 안전하게

이미 규격에 맞는 원고는 100% 원본 그대로 유지하고, 규격을 넘치는 원고만 재단선 안쪽으로 필요한 만큼만 줄여 글자가 잘리지 않게 합니다.

판권지 자동 구성

교보 표준 판권지에 책 제목·발행일·지은이만 정확히 교체. 앞/뒤/없음 위치까지 선택.

반려 방지 설계

여백 없이 판형에 딱 맞게, 재단여백 3mm까지 컬러/이미지로 꽉 채워 검수 반려를 예방합니다.

판권 자동 생성

교보 공식 판권 양식 그대로. 제목·저자·발행일만 넣으면 본문 앞이나 뒤에 자동으로 붙습니다. 글꼴도 고를 수 있어요.

규격 자가검증

추출 직전 완성된 PDF의 실제 페이지 크기를 다시 읽어 교보 규격과 대조합니다. 전 페이지 일치 여부를 리포트로 확인하고 안심하고 올리세요.

내 원고는 내 컴퓨터에만 — 100% 로컬 저장

저작권 보호와 유출 방지를 위해, 원고 PDF 등 모든 창작물은 서버에 업로드되지 않고 브라우저(로컬)에서만 처리됩니다. 서버에는 로그인·결제·이용통계만 안전하게 보관돼요. 파일 생성·추출도 전부 사용자의 브라우저 안에서 처리됩니다.

교보문고 바로출판 POD

원고 PDF를 원하는 책 규격으로.

한글·워드에서 만든 원고를 PDF로 저장해서 올려주세요. 원하는 판형만 고르면 교보 규격에 100% 맞춰 내용을 재단선 안쪽으로 안전하게 조정하고, 판권까지 넣어 바로 업로드할 내지 PDF를 만들어드려요.

1

내지 원고 PDF 올리기 PDF만

📄
내지 원고 PDF를 여기로 끌어다 놓기 (또는 클릭)
PDF 파일만 업로드 가능해요 · 브라우저 로컬 처리
💡 원고는 왼쪽=짝수 / 오른쪽=홀수 순서, 총 페이지 수는 짝수로 준비해 주세요.
한글(hwp)·워드(docx) 파일은 어떻게 올리나요?
hwp·docx는 직접 올릴 수 없어요. 원본 프로그램에서 PDF로 저장한 뒤 그 PDF를 올려주세요. 원본 프로그램이 직접 그린 PDF라 레이아웃이 가장 정확합니다.

· 한글(HWP) — 파일 > PDF로 저장하기
· MS 워드 — 파일 > 다른 이름으로 저장 > 파일 형식 PDF
· 구글 독스 — 파일 > 다운로드 > PDF 문서(.pdf)
· 페이지스(Mac) — 파일 > 내보내기 > PDF

💡 팁: 저장 전에 용지 크기를 원하는 판형(예: 신국판 152×225mm)으로 맞춰두면, 축소 없이 글자 크기가 100% 그대로 유지돼요.
2

책 규격(판형) 선택 원고에서 자동 감지 · 바꿀 수 있어요

3

판권 정보 판권 페이지에 들어갈 내용

📖 페이지 번호는 알아서 처리해요
원고를 분석해 이미 번호가 있으면 그대로 두고, 없으면 교보 규칙대로 자동으로 넣어드려요 (홀수=오른쪽 / 짝수=왼쪽 바깥쪽). 본문과 겹치지 않게 하단 공간을 확보하니 안심하세요.
