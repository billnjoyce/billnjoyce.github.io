세모(SEMO) 브랜드 사이트
========================

[파일]
- index.html : 단일 자체완결 파일. 모든 이미지가 내장(base64)되어 있어
  추가 파일 없이 브라우저로 바로 열면 됩니다. 웹 서버 업로드 시에도 이 파일 하나면 동작.
- assets/    : (선택) 사이트에 사용된 원본 이미지. 교체/편집용.

[스토어 링크]
- App Store : https://apps.apple.com/app/id6770508450
- Google Play : https://play.google.com/store/apps/details?id=com.joycestudios.semo

[구성]
히어로 → 기능4(정모/세차장/상세/만들기) → 세차 날씨(다크) → 화면 갤러리 → 가치 3카드 → 다운로드 CTA → 푸터

[수정 포인트]
- 푸터의 '이용약관/개인정보처리방침' 링크(href="#")를 실제 URL로 교체
- 문의 메일(support@semo.app) 확인
- 폰트는 시스템 폰트(애플기기=SF/Apple SD Gothic, 윈도우=맑은고딕) 사용 — 외부 의존 없음
